Demo: Adjust Smart Banner Integration

Overview
This project is a demo implementation of an Adjust Smart Banner integration created for testing and learning purposes.
The goal of this repository is to simulate how an Adjust Smart Banner behaves on a website and to understand the end-to-end user journey from:
Website Visit → Smart Banner Display → Banner Click → Deep Link / Store Redirect

[Important]: This implementation does not communicate with production Adjust services and should not be used in a live environment.

Objectives:
This project was created to:

Understand the Smart Banner integration flow
Simulate app-open and app-install journeys
Test deep linking behaviour
Validate redirect logic
Demonstrate banner rendering and user interaction
Learn how Adjust tracking links are used within Smart Banners

User Flow
User visits website -> Smart Banner displayed -> User clicks banner -> Adjust tracking link triggered -> App Installed? (Based on the APP integration) Deeplinked or Store redirection.
